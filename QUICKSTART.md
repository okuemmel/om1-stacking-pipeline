# Quick Start Guide

Get stacking in 5 minutes! ⚡

## 1. Download & Install (30 seconds)

```bash
# Download latest DMG
# From: https://github.com/okuemmel/om1-stacking-pipeline/releases

# Open DMG and drag app to Applications
```

## 2. Install Dependencies (2 minutes)

```bash
# Install Homebrew (if not installed)
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Install tools
brew install imagemagick exiftool

# Download Helicon Focus
# https://www.heliconsoft.com/heliconsoft-products/helicon-focus/
```

## 3. First Launch (1 minute)

```bash
# Right-click app → Open (first time only)
# Browser opens automatically at http://localhost:8080
```

## 4. Stack Your First Series (2 minutes)

1. **Insert SD card** with focus-bracketed images
2. **Click "Scan for SD Cards"** in browser
3. **Select your SD card**
4. **Review series** (thumbnails generated automatically)
5. **Click "Start Stacking"**
6. **Watch progress** in real-time
7. **Done!** Find stacks in `~/Pictures/Stacked/`

## 🎉 That's it!

**Total time: ~5 minutes**
